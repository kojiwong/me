PROJECTS

Here are some of my current and past projects.

User-level Threads Package

c, linux

For my operating systems course, I developed a user-level threads package to allow for tasks to run concurrently within a process. I utilized timer interrupts and signals for preemptively switching threads and created mutual exclusion and synchronization primitives for locks and condition variables. The list of implemented thread functions include create, yield, kill, exit, sleep, wakeup, wait, where the picture above is the schematic for the thread_wait function.

COVID-19 CT Scan Image Classification

python, scikit-learn, tensorflow, matplotlib

I developed a Convolutional Neural Network (CNN) with two others to predict if a patient has COVID-19 based on CT scans of their lungs. To assess our model, we analyzed metrics such as accuracy, recall, and F1-score.

Sokoban Optimal Solver

python

I developed a solver for the Sokoban puzzle for my Artificial Intelligence class, where the objective is to place all boxes onto specific spots. This solver was implemented with the A* algorithm which finds the shortest path and involved developing an advanced heuristic in order to find the optimal solution.
